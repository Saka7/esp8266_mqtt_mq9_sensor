# ESP8266 MQTT MQ9 sensor

Board: ESP8266
OS: FreeRTOS
SDK: ESP8266_RTOS_SDK
Sensor: MQ9
MQTT topic: 'mq9'

## Setup

Follow ESP8266_RTOS_SDK setup guide [link](https://github.com/espressif/ESP8266_RTOS_SDK)

## Commands

### Compile and flash

```sh
make app-flash
```

### Show serial monitor logs

```sh
make monitor
```

> Ctrl+] to exit
